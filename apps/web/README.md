## Contribute to project

1. Clone repo:

    ```bash
    git clone https://github.com/Sports-Recruitment-Platform/frontend.git
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Create and switch to your own branch:

    ```bash
    git checkout -b your-branch-name
    ```

4. Make your changes and commit them:

    ```bash
    git add .
    git commit -m "Your commit message"
    ```

5. Push the branch to the remote repository:

    ```bash
    git push -u origin your-branch-name

    # run only `git push` for subsequent pushes to remote
    ```

## Vite + React + TypeScript

The React project is setup using Vite as the framework and TypeScript as the language for type checking.

Eslint has also been configured for tsx, jsx support and TypeScript type checking.

## Libraries and dependencies

-   UI styling - TailwindCSS and Shadcn UI.
-   Icons library - Lucide React.
-   Client-side routing - Remix React router.
-   Making HTTP requests and server-side state management - Axios and React Query.
-   Form management - React-hook-form and Zod.
-   Animations library - GSAP React.
